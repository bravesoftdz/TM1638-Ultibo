# TM1638-Ultibo

TM1638
======

Controlling TM1638 modules with Ultibo Raspberry Pi

Connecting the TM1638 to Raspberry Pi:

<pre>
RPi               TM1638 based board
3.3V    ------------------ VCC
GND     ------------------ GND
PIN #23 ------------------ STB
PIN #17 ------------------ DIO
PIN #22 ------------------ CLK
</pre>
